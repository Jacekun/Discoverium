# ![Discoverium Icon](./assets/graphics/icon_small.png) Discoverium

Get Android app updates straight from the source.

Discoverium allows you to install and update apps directly from their releases pages, and receive notifications when new releases are made available.

More info:
- [Discoverium Wiki](https://wiki.obtainium.imranr.dev/) ([repository](https://github.com/cygnusx-1-org/Discoverium-Wiki))
- [Discoverium 101](https://www.youtube.com/watch?v=0MF_v2OBncw) - Tutorial video
- [AppVerifier](https://github.com/soupslurpr/AppVerifier) - App verification tool (recommended, integrates with Discoverium)
- [apps.obtainium.imranr.dev](https://apps.obtainium.imranr.dev/) - Crowdsourced app configurations ([repository](https://github.com/cygnusx-1-org/apps.obtainium.imranr.dev))
- [Side Of Burritos - You should use this instead of F-Droid | How to use app RSS feed](https://youtu.be/FFz57zNR_M0) - Original motivation for this app
- [Website](https://obtainium.imranr.dev) ([repository](https://github.com/cygnusx-1-org/obtainium.imranr.dev))
- [Source code](https://github.com/cygnusx-1-org/Discoverium)

Currently supported App sources:

  - [SourceHut](https://git.sr.ht/)
- Other - General:
  - [APKPure](https://apkpure.net/)
  - [Aptoide](https://aptoide.com/)
  - [Uptodown](https://uptodown.com/)
  - [Huawei AppGallery](https://appgallery.huawei.com/)

  - [CoolApk](https://coolapk.com/)

- Other - App-Specific:
  - [Telegram App](https://telegram.org/)
  - [Neutron Code](https://neutroncode.com/)
- Direct APK Link
- "HTML" (Fallback): Any other URL that returns an HTML page with links to APK files

## Finding App Configurations

You can find crowdsourced app configurations via the search button in the app.

If you can't find the configuration for an app you want, feel free to leave a request on the [discussions page](https://github.com/cygnusx-1-org/Discoverium/issues).

## Limitations
- For some sources, data is gathered using Web scraping and can easily break due to changes in website design. In such cases, more reliable methods may be unavailable.
